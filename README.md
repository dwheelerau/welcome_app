# welcome_app
First app.

## Install  
```
# only needed once
npm i -g parcel-bundler

# initilise the project
yarn init -y

# add react
yarn add react react-dom
```

Next modify the `package.json` file to add a script section
```
  "scripts": {
    "start": "parcel ./src/index.html --open"
  },
```
